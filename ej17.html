Definition list

API ( Application Programming Interface )
Interfaz de programación de aplicaciones. Una serie de rutinas usadas por una aplicacion para gestionar generalmente, servicios de bajo nivel, realizados por el sistema operativo de la computadora.
Bug
Término aplicado a los errores descubiertos al ejecutar cualquier programa informático.
Commit
Consolidar, confirmar o hacer un commit se refiere a la idea de confirmar un conjunto de cambios provisionales de forma permanente.
